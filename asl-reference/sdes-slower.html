sdes-slower

Signature

TimeSeriesExpr -- TimeSeriesExpr

Summary

Helper for computing sliding DES using settings to quickly adjust to the input line. See recommended values for more information.

Example

/api/v1/graph?
  q=
    name,requestsPerSecond,:eq,
    :sum,
    :sdes-slower
