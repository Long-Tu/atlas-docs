random

Signature

 -- TimeSeriesExpr

Summary

Generate a time series that appears to be random noise for the purposes of creating sample data to experiment with expressions. To ensure that the line is deterministic and reproducible it actually is based on a hash of the timestamp. Each datapoint is a value between 0.0 and 1.0.

See also :srandom.

Example

/api/v1/graph?
  q=
    :random
